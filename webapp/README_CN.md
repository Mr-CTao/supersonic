## 标签能力开关

标签市场、标签对象管理、维度/指标“设为标签”等能力默认关闭。二次开发如果依赖标签体系，可以使用以下命令启动或构建：

```bash
cd packages/supersonic-fe

# 开发模式打开标签能力
pnpm dev:os:tag

# 构建时打开标签能力
pnpm build:os:tag
```

也可以在已有命令前添加 `SHOW_TAG=true`。开启后请验证“创建主题域 -> 创建标签对象 -> 创建模型并绑定标签对象 -> 将维度或指标设为标签 -> 标签市场查询”的完整链路。
